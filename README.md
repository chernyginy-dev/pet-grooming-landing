# Pets Grooming Landing Page

A modern responsive landing page for a pet grooming service.
Created as a frontend portfolio project demonstrating clean layout structure, responsive design, and user-friendly interface development.

🔗 **Live Demo:**
https://chernyginy-dev.github.io/pet-grooming-landing

---

## Overview

This project represents a business landing page designed for a pet grooming service.
The layout highlights services, pricing, and customer benefits while maintaining a friendly and approachable visual style suitable for a pet care brand.

The page focuses on clear structure, strong visual hierarchy, and a smooth browsing experience across devices.

---

## Features

- Responsive layout
- Modern and clean UI
- Structured HTML markup
- Mobile-friendly navigation
- Service-focused landing page structure
- Cross-browser compatibility
